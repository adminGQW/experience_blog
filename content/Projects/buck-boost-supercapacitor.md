## 项目简介

为解决 RoboMaster 机器人底盘低速时功率冗余、重载时瞬态功率不足的问题，先后开发了两代超级电容控制器。V1 基于桂林理工大学开源方案 (STM32G431)，V2 参考香港科技大学方案并重新用 C 重写固件 (STM32F334)，最终实现功率闭环 + 缓冲能量闭环的智能功率管理。

## 技术栈

- **V1 主控：** STM32G431CBT6 (ARM Cortex-M4, 170MHz)
- **V2 主控：** STM32F334 (高分辨率定时器 HRTIM)
- **拓扑结构：** 双向 Buck-Boost (V1: 半桥 MOS; V2: 四开关 Buck-Boost)
- **储能元件：** 法拉电容串联组
- **PCB 设计：** KiCad 8.0
- **通信：** CAN / FDCAN

## 项目时间

2024.09 - 2025.04

## 核心功能

1. **双向 DC-DC 变换：** Buck 模式充电储能，Boost 模式放电输出，根据裁判系统实时功率自动切换
2. **多环 PID 控制：** 电压环 (防过压) + 电流环 (防过流) + 功率环 (恒功率充放电)，双环竞争取最小值输出
3. **功率闭环 + 缓冲能量闭环：** 底盘输出经功率限制，超级电容在功率裕余时储能、功率不足时释放，实测比赛全程零超功率
4. **CAN 通信协议：** 接收电控使能/充电/功率上限指令，上报超电能量百分比/实时功率/状态标志，200ms 离线检测 + 方波重连
5. **多重安全保护：** 软启动 → 过压/欠压/过流/过温保护 (8 种状态机) + CAN 看门狗 + 断电检测

## 技术亮点

- **V1 双环竞争输出：** 电压环和电流环取最小值作为 PWM 占空比，确保任何时刻不会同时超压和超流；ADC 采样在 PWM 周期内精确触发以避开开关噪声
- **V2 四开关平滑过渡：** 利用 STM32F334 的 HRTIM 实现 288kHz 开关频率，Buck / Boost / Buck-Boost 三种模式平滑过渡，峰值效率约 96%，保障最高 360W 瞬态输出
- **V2 分离式热管理：** 功率电路铝基板与 4 层 FR4 控制板上下分离，优化散热
- **C++ → C 重写：** V2 原版用 C++ 编写，可读性和通用性对本队较差；完整重写为 C 并适配实际板子参数，修复原版参数不匹配问题

## 踩坑记录

- **INA240PW1 vs PW2：** 第二块 V2 板子充电异常，排查发现底盘电流检测芯片选型错误 — INA240PW2 增益与 PW1 不同，更换后正常
- **电容炸裂：** V2 调试期间发生过电容损坏，需严格设置软件过压保护阈值
- **CAN 离线恢复：** 设计了 1s-on / 1s-off 方波重连机制，连续收到 10 次充电指令才判定在线，防止误触发
